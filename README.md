Leads Tracker — Chrome Extension

Never lose track of a website again. Leads Tracker is a lightweight Chrome extension that lets you save URLs instantly while browsing — either by typing them in manually or saving your current tab with one click.


Features


Save any URL manually — type or paste a link and save it to your list
Save current tab instantly — no copying and pasting, just click Save Tab
Persistent storage — your leads are saved to localStorage, so they survive closing the browser
One-click delete — clear your entire list when you're done



How to Use

Save a URL manually


Open the extension by clicking its icon in the Chrome toolbar
Type or paste a URL into the input field (e.g. www.xyz.com)
Click Save Input — it appears in your list instantly


Save your current tab


Navigate to any webpage you want to save
Open the extension
Click Save Tab — the current page URL is added to your list automatically


Visit a saved link

Click any link in your list — it opens the URL directly

Clear your list

Click the Delete button to wipe all saved links and start fresh

Installation (Developer Mode)

Since this extension isn't on the Chrome Web Store, load it manually:


Download or clone this project folder to your computer
Open Chrome and go to chrome://extensions
Toggle on Developer mode (top right)
Click Load unpacked
Select the project folder
The Leads Tracker icon will appear in your Chrome toolbar



Who Is This For?

This extension is handy for:

Job seekers saving company or job listing URLs while researching
Recruiters tracking candidate or client websites
Developers bookmarking references, docs, and tools while building
Researchers collecting sources across multiple tabs
Anyone who needs a quick, no-fuss way to save links on the fly
